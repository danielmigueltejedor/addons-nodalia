# Soporte y diagnostico

## Que incluir en un reporte

- Version de Home Assistant.
- Version exacta de la app (estable/beta).
- Logs relevantes (supervisor + app).
- Capturas o error textual.
- Pasos para reproducir.

## Tailscale: chequeos utiles

- Estado backend (`Running`, `NeedsLogin`, etc.).
- WebUI local disponible o no.
- ACL/tagOwners correctos para control remoto.
- Si aplica, estado del soporte remoto.

## Donde pedir ayuda

- Issues del repositorio: `apps-nodalia`.
