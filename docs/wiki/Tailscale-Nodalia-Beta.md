# Tailscale (Nodalia Beta)

Canal beta de Tailscale con cambios en validacion.

## Para que usar beta

- Probar mejoras antes de estable.
- Detectar regresiones pronto.
- Validar UX y rendimiento del panel.

## Buenas practicas al probar

1. Probar instalacion limpia.
2. Probar actualizacion desde version anterior.
3. Adjuntar logs de supervisor y app.
4. Indicar tiempo de carga de WebUI y pasos exactos.
